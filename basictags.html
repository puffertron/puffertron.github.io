Anchor tags <a> are used to link to other pages or to external websites

The destination address of the link is set using the href attribute

        
<a href="https://puffertron.net">Puffertron.Net</a>
<a href="https://notneelpatel.github.io">Neel Patel's Website</a>

List tags are used to create lists (duh). An unordered list tag <ul> will create an unordered list. An ordered list tag <ol> will create an ordered list.

To define items in a list, we use the list item tag <li> tag for each item in the list


        

<p>Unordered list</p>
<ul>
    <li>thing one</li>
    <li>thing two</li>
    <li>thing three</li>
</ul>

Unordered list

  • thing one
  • thing two
  • thing three
Ordered list (best anime)

  3. Neon Genesis Evangelion
The Image tag <img> is used to create images. We define the source of our image with the src tag. If your image is too big, you can restrict its size with the width or height attribute. 
        
<img src="https://link.to.image" width=500>
<img src="/path/to/local/image.png" width=500>

an image of a weird lookin dog
